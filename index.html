CVCS vs DVCS

CVCS (centralized version control system) such as Subversion (SVN) store all the code and history in a central repository. On the other hand, DVCS (distributed version control system) like Git stores a copy of the entire repository on each developer's computer, making it easier to work offline and merge changes between repositories.

Getting Started with Git

To get started with Git, open the terminal and use the following commands:

  • ls - list files and directories in the current directory
  • cd - change directory

There are two ways to start a project with Git:

  • Create a repository and start adding files
  • Create a project and initialize a Git repository in it

If you are a new user, you need to configure your Git username and email using the following commands:

  • git config --global user.name "Your Name"
  • git config --global user.email "your_email@example.com"

To initialize a Git repository in your project directory, use the following command:

  • git init - initialize Git

You can use the ls -a command to see all the hidden files in the current directory. The tree command can be used to view the folder structure. To see all the files and folders in the Git repository, use the following command:

  • tree .git/ - view the Git repository structure

The git status command shows you the changes made in your project. To add a file to Git, use the following command:

  • git add filename - add a file to Git (the data is added to the cache and a commit ID is generated)

After adding files to Git, you can create a commit using the following command:

  • git commit -m "Commit message" - create a commit with a message as an identifier for each version

Each commit generates a unique version ID. To view all the commit versions, use the following command:

  • git log - view all commit versions

You can also move to a specific commit using the git checkout command and the commit ID. To view all commits, including those that have been removed, use the following command:

  • git log --all - view all commits
